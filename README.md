# Basket-Game
this is my first mini project in python using GUI
🎮 Catch the Falling Objects Game
A simple Python mini project built with Tkinter where the player controls a basket to catch falling objects. This project demonstrates key programming concepts like coordinates, animation, keyboard input, and collision detection.

✨ Features
🧺 Basket controlled with Left/Right arrow keys

🍎 Random falling objects generated at the top of the window

⚡ Smooth animation using Tkinter’s after() method

🎯 Collision detection to check if the basket catches the object

📊 Score counter that increases when you catch an object

🛠 Concepts Practiced
Coordinates: Representing positions of basket and objects with (x1, y1, x2, y2)

Animation: Moving objects continuously using after()

Keyboard Input: Binding arrow keys to move the basket

Collision Detection: Checking overlap between basket and object

Game Loop: Repeatedly updating positions and score
